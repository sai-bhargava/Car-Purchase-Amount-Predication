🚗 Car Purchase Amount Prediction using ANN
📌 Project Overview

This project aims to predict the purchase amount of a car based on customer attributes using an Artificial Neural Network (ANN) model. The goal is to build a regression model that learns patterns from historical data and accurately estimates how much a customer is likely to spend on a car.

🎯 Objective
Develop a machine learning model using ANN to predict car purchase amounts
Analyze the impact of customer features such as income, age, and credit score
Improve prediction accuracy using deep learning techniques
🧠 Model Used
Artificial Neural Network (ANN)
Input Layer: Customer features
Hidden Layers: Dense layers with activation functions (ReLU)
Output Layer: Single neuron for regression output
📊 Dataset Features

Typical features used in the dataset include:

Customer Name / ID
Gender
Age
Annual Salary
Credit Card Debt
Net Worth
